## Features {#features}

-   基于注解的Spring Boot编程模型
-   用于将查询测试编写为单元测试的测试框架
-   Gradle 代码生成插件用于从模式创建类型
-   与 GraphQL Federation 轻松集成
-   与 Spring Security 集成
-   GraphQL 订阅（WebSocket 和 SSE）
-   文件上传
-   错误处理
-   许多扩展点
